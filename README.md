# Marksy

**Marksy** is a tool that allows you to turn your browser bookmarks into a public and organized directory. Through a browser extension, you can automatically sync your links to a personalized page where you can organize them into categories, assign tags, and control their privacy settings.

🚧 **Currently in Development** 🚧
Marksy is still under construction. The main page ([Marksy.co](https://marksy.co/)) currently features a landing/waitlist page, and you can check out a demo of the directory at [andryore.marksy.co](https://andryore.marksy.co/). Some features are not yet available.

## Features 🌟

- 🔄 **Automatic Syncing**: Each new bookmark is synchronized with your public directory. _(Coming Soon)_
- 🔒 **Privacy Control**: Choose which bookmarks are public and which remain private.
- 📂 **Advanced Organization**: Bookmarks are structured according to your browser folders. _(Coming Soon)_
- 🏷 **Tags and Filters**: Make searching and navigation easier within your directory.
- 🎨 **Customization**: Add social media links, choose a visual theme, and personalize your profile.
- 📌 **Intuitive Dashboard**: Manage your bookmarks and adjust settings from a simple interface. _(Coming Soon)_

## Installation 🔧

1. Clone the repository:

```bash
 git clone <your-repo-url>
 cd marksy
```

2. Install dependencies:

```bash
 pnpm install
```

3. Start the development server:

```bash
 pnpm dev
```

4. Build the application for production:

```bash
 pnpm build
```

## Usage 📚 _(Coming Soon)_

1. **Sign up** at [Marksy.co](https://marksy.co/) and set up your profile.
2. **Install the extension** in your browser and authenticate.
3. **Select** which folders you want to sync.
4. **Access your public directory** (e.g., `https://username.marksy.co`).
5. **Customize and organize** your bookmarks from the dashboard.

📢 **Currently, you can only join the waitlist at [Marksy.co](https://marksy.co/) and view a demo at [andryore.marksy.co](https://andryore.marksy.co/).**
